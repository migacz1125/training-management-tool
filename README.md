### Install commands
  `npm install`

  `jspm install`

  `typings install`

### Dev server run command
`npm start`

##### Tips
- open browser: localhost:8080/src/
- index page: src/index.html
- use JSPM beta -> `jspm@0.17.X`
- follow BEM conventions: https://en.bem.info/methodology/naming-convention/
